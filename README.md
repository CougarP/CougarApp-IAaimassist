# CougarApp

## Purpose

CougarApp is an application developed for educational purposes. It demonstrates advanced programming techniques, image processing, input device control, and the use of machine learning libraries. This application should not be used in online games or any other activity that violates the terms of service of any service or software.

## Educational Use

This software is designed to illustrate concepts such as:
- Use of CUDA libraries for GPU acceleration
- Real-time image processing
- Implementation of PID control algorithms
- Manipulation of input devices like mouse and keyboard

## Features

- **Aim Assistance:** Automatically adjusts the aim to target objects detected in the game.
- **Recoil Control:** Reduces the recoil effect when firing in the game, making shooting more accurate.
- **Visualization Window for Debugging:** Provides a real-time display of what the software is processing, useful for debugging and development.
- **Image Saving and Annotation:** Captures and saves images with annotations, which can be used to create your own dataset for machine learning purposes.
- **Various Configurations:** Offers multiple settings and profiles that can be customized to suit different needs and scenarios.
